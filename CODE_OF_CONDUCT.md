# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All participants are expected to uphold it. Enforcement inquiries should be directed through [GitHub Private Vulnerability Reporting](https://github.com/lopes/lantana/security/advisories/new) — this is a single-maintainer project with no separate moderation team.
